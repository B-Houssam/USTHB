Le but de ce TP est de modifier les attributs du DOM afin de créer une alternance de deux images à intervalle régulier. Pour cela, vous allez créer une page html contenant une image, et programmer en JavaScript une alternance entre cette image et une autre, toutes les deux secondes. Enfin nous allons ajouter deux boutons : un bouton pour arrêter ce défilement et un bouton pour le reprendre.

L’usage des nouvelles fenêtres, appelées aussi fenêtres popup, est souvent utile pour afficher des informations complémentaires sans surcharger la page de départ. Elles comportent cependant des inconvénients : ces fenêtres sont utilisées de manière abusive à des fins publicitaires et donc bloquées par les navigateurs pour protéger l’utilisateur. Il est conseillé de bien fermer les fenêtres popup et à les utiliser avec modération.

Créer une page HTML avec un formulaire et un bouton. Le clic sur le bouton permet de créer une nouvelle fenêtre permettant d’afficher le fichier bonjour.txt créé précédemment. Spécifier pour cette fenêtre une hauteur et une largeur de 500, positions haut et gauche de 100 chacune.
